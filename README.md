EduTrack Lite

EduTrack Lite is a lightweight student result management system designed to help schools track academic performance efficiently. 
🌍 Problem

Many small schools and private tutors manage student results manually using paper or spreadsheets. This often leads to errors, poor record keeping, and lack of performance tracking.

👥 Users

Secondary schools

Private tutors

Educational centers

School administrators

🛠️ Features

Add student details

Select class, term, and subject

Automatic grade calculation

Prevent duplicate subject entries

Total and average score calculation

Performance chart visualization

Printable report card

Dark mode toggle

Responsive dashboard layout

LocalStorage data persistence

💻 Technologies Used

HTML5

CSS3

JavaScript

Chart.js

🚀 How to Run

Clone the repository

Open index.html in your browser

Start adding student results

📈 Impact

EduTrack Lite simplifies academic result management and improves transparency in student performance tracking.
